This reference catalogs every interactive element discovered across the OpenBat platform, organized by element type.

## Modals

| Modal | Location | Fields |
|-------|----------|--------|
| Create a chatbot | `/platform` > **+ New chatbot** | Chatbot name (text, required, 0/100 character limit) |
| Invite member | `/platform/members` > **+ Invite member** | Email (text, required), Role (dropdown: Admin or Member) |

## Tabs

Tabs organize content across many pages:

| Page | Tabs |
|------|------|
| Dashboard (chatbots) | Active, Pending, Archive |
| Analytics dashboard | User Insights, Assistant Performance |
| Workflows | Workflows, Runs, Templates |
| Analysis config | Metadata, Translation, User Analysis, Assistant Analysis, Prompts |
| Chatbot settings | General, Company Info, API Keys, Members, Webhooks, Import/Export Data |

## Dropdowns

### Date range picker

Available on the analytics dashboard, conversations, deep search, and organizations pages. Presets: Today, Yesterday, This week, Last week, This month, Last month, This year, Last year, All time, Custom (dual calendar picker).

### Segment by

Analytics dashboard dropdown to break down metrics by: Plan, Industry, Device, or Country.

### Filter popovers

Available on the conversations, organizations, and users pages. Filter fields include:

| Field | Available on |
|-------|-------------|
| User Industry | Conversations, Users |
| Org Plan | Conversations, Organizations |
| Device | Conversations |
| Language | Conversations |
| Template name | Conversations |
| Messages (count range) | Conversations |
| Sentiment average (score range) | Conversations |
| User MRR | Conversations, Users |
| Org MRR | Conversations, Organizations |

### Column visibility toggle

Click the **View** button on table pages (conversations, organizations, users) to show or hide columns. The conversations table has 11 toggleable columns.

### Pagination

Table pages offer rows per page: 10, 20, 50, or 100.

### Other dropdowns

| Dropdown | Location | Options |
|----------|----------|---------|
| Role | Invite member modal | Admin, Member |
| Export format | Settings > Import/Export Data | JSON |
| Industry | Settings > Company Info | Developer Tools, etc. |
| Primary language | Analysis Config > Translation | English, etc. |

## Context menus

### Chatbot card menu

Click the **...** button on a chatbot card on the main dashboard:

<Frame>
  <img src="/docs/state-screenshots/state-platform-chatbot-context-menu.png" alt="Chatbot context menu with Open chatbot, Archive, and Delete options" />
</Frame>

| Action | Description |
|--------|-------------|
| Open chatbot | Navigate to the chatbot's analytics dashboard |
| Archive | Move to the Archive tab (stops receiving conversations) |
| Delete | Permanently remove the chatbot and all data |

| Login | Password | text | Yes | — |

### Platform forms

| Page | Field | Type | Required | Validation |
|------|-------|------|----------|------------|
