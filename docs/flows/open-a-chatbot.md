# Open a chatbot

## Overview

| Property | Value |
|----------|-------|
| **Flow** | Open a chatbot |
| **Starting Page** | `/platform` |
| **Persona** | Analyst reviewing chatbot performance data |

## Description

Navigate from dashboard to a chatbot's analytics page

## Preconditions

- User is logged in
- At least one chatbot exists

## Steps

### Step 1

Navigate to /platform and see chatbot cards

### Step 2

Click '...' menu on chatbot card

### Step 3

Click 'Open chatbot' from context menu

### Step 4

Verify chatbot dashboard loads with KPIs, sentiment charts, and analysis data

## Expected Outcome

User sees chatbot dashboard with KPIs (373 conversations, 1292 messages), sentiment chart, frustrations, dissatisfied users, topics, revenue signals, intents, heatmap, languages
