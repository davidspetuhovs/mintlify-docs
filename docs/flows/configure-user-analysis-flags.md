# Configure user analysis flags

## Overview

| Property | Value |
|----------|-------|
| **Flow** | Configure user analysis flags |
| **Starting Page** | `/platform/{chatbotId}/analysis-config` |
| **Persona** | Product manager customizing signal detection |

## Description

Toggle and customize analysis flags for user messages

## Preconditions

- User is logged in
- User has access to analysis config

## Steps

### Step 1

Navigate to analysis config — tabs: Metadata, Translation, User Analysis, Assistant Analysis, Prompts

## Expected Outcome

User Analysis tab allows toggling flags and creating custom ones
