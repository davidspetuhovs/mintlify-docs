# Create a new chatbot

## Overview

| Property | Value |
|----------|-------|
| **Flow** | Create a new chatbot |
| **Starting Page** | `/platform` |
| **Persona** | Developer or admin setting up OpenBat for a new project |

## Description

Create a new chatbot from the dashboard

## Preconditions

- User is logged in
- Organization has available chatbot slots

## Steps

### Step 1

Navigate to /platform

### Step 2

Click '+ New chatbot' button, modal opens with name field

### Step 3

Enter chatbot name and click 'Continue to setup'

## Expected Outcome

Chatbot created, redirected to 6-step onboarding wizard (Basics, Product, Brand, Pricing, Context, Connect)
