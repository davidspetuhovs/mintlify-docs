# Enable auto-translation

## Overview

| Property | Value |
|----------|-------|
| **Flow** | Enable auto-translation |
| **Starting Page** | `/platform/{chatbotId}/analysis-config` |
| **Persona** | Product team with international users |

## Description

Enable automatic translation of non-primary-language messages

## Preconditions

- User is logged in
- Chatbot receives messages in multiple languages

## Steps

### Step 1

Click Translation tab in analysis config

## Expected Outcome

Translation settings with toggle and language selection
