# Configure API keys

## Overview

| Property | Value |
|----------|-------|
| **Flow** | Configure API keys |
| **Starting Page** | `/platform/{chatbotId}/settings` |
| **Persona** | Developer setting up or maintaining SDK integration |

## Description

View and rotate API keys for SDK authentication

## Preconditions

- User is logged in
- User has access to chatbot settings

## Steps

### Step 1

Navigate to settings, click API Keys tab — shows masked key prefix ob_live_8ae071a6 with copy and Rotate key buttons

## Expected Outcome

API key visible (masked) with copy and rotate functionality
