# Configure webhooks

## Overview

| Property | Value |
|----------|-------|
| **Flow** | Configure webhooks |
| **Starting Page** | `/platform/{chatbotId}/settings` |
| **Persona** | Operations lead setting up Slack/Discord alerts |

## Description

Add webhook destinations for workflow alerts

## Preconditions

- User is logged in
- User has a webhook URL

## Steps

### Step 1

Click Webhooks tab — empty state with 'Add webhook' button and docs for Discord, Slack, and Custom types

## Expected Outcome

Webhook destination saved and available for workflow actions
