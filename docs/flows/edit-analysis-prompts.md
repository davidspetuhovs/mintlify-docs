# Edit analysis prompts

## Overview

| Property | Value |
|----------|-------|
| **Flow** | Edit analysis prompts |
| **Starting Page** | `/platform/{chatbotId}/analysis-config` |
| **Persona** | Power user or ML engineer fine-tuning analysis |

## Description

Customize the AI prompts used for conversation analysis

## Preconditions

- User is logged in
- User understands prompt engineering

## Steps

### Step 1

Click Prompts tab in analysis config

## Expected Outcome

Prompt editor with Sentiment, Intent, Topics, Flags cards
