# Export conversation data

## Overview

| Property | Value |
|----------|-------|
| **Flow** | Export conversation data |
| **Starting Page** | `/platform/{chatbotId}/settings` |
| **Persona** | Data analyst running custom analysis or admin creating backup |

## Description

Download conversations and analysis results as JSON

## Preconditions

- User is logged in
- Chatbot has conversations

## Steps

### Step 1

Click Import/Export Data tab — Export section with JSON format selector, 'Include analysis results' checkbox, '1,292 messages' count, Export button

## Expected Outcome

JSON file with all conversations and analysis results downloads
