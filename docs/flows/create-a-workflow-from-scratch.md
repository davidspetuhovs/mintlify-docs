# Create a workflow from scratch

## Overview

| Property | Value |
|----------|-------|
| **Flow** | Create a workflow from scratch |
| **Starting Page** | `/platform/{chatbotId}/workflows` |
| **Persona** | Power user who needs custom automation |

## Description

Create a custom workflow with name, conditions, and actions

## Preconditions

- User is logged in
- Organization's plan allows workflows

## Steps

### Step 1

Click 'Create workflow' button on Workflows tab — inline form appears with name field and Create/Cancel buttons

## Expected Outcome

Workflow created and available for configuration
