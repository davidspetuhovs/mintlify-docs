# Import conversation data

## Overview

| Property | Value |
|----------|-------|
| **Flow** | Import conversation data |
| **Starting Page** | `/platform/{chatbotId}/settings` |
| **Persona** | Developer with historical conversation logs |

## Description

Upload historical conversations for analysis

## Preconditions

- User is logged in
- User has conversation data in JSON or CSV

## Steps

### Step 1

Import section on Import/Export Data tab — 'Download template CSV' button, file chooser, Import button (disabled until file selected)

## Expected Outcome

Conversations imported and available for analysis
