# Browse client organizations

## Overview

| Property | Value |
|----------|-------|
| **Flow** | Browse client organizations |
| **Starting Page** | `/platform/{chatbotId}/organizations` |
| **Persona** | Customer success manager monitoring account health |

## Description

View and filter customer organizations with sortable metrics

## Preconditions

- User is logged in
- Chatbot has conversations with organization metadata

## Steps

### Step 1

Navigate to organizations page — 48 rows with columns: Organization, Plan, Industry, MRR, Conversations, Users, Sentiment, Last Activity

## Expected Outcome

Sortable, filterable table of customer organizations
