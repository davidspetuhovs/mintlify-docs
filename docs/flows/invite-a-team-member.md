# Invite a team member

## Overview

| Property | Value |
|----------|-------|
| **Flow** | Invite a team member |
| **Starting Page** | `/platform/members` |
| **Persona** | Organization admin who wants to give team members access |

## Description

Invite a new member to the organization

## Preconditions

- User is logged in
- User has Owner or admin role

## Steps

### Step 1

Navigate to /platform/members and see members table

### Step 2

Click 'Invite member' button — modal opens with email and role fields

### Step 3

Verify role dropdown has Admin and Member options

## Expected Outcome

Invitation sent to email, member appears in table once accepted
