# Delete organization

## Overview

| Property | Value |
|----------|-------|
| **Flow** | Delete organization |
| **Starting Page** | `/platform/settings` |
| **Persona** | Organization owner who wants to remove the organization |

## Description

Permanently delete the organization and all data

## Preconditions

- User is logged in
- User has Owner role

## Steps

### Step 1

Navigate to /platform/settings

### Step 2

Click 'Delete organization' in danger zone

### Step 3

Confirmation dialog requires typing org name to confirm — Delete button disabled until matched

## Expected Outcome

Organization and all data permanently deleted
