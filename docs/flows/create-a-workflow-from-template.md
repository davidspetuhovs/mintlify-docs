# Create a workflow from template

## Overview

| Property | Value |
|----------|-------|
| **Flow** | Create a workflow from template |
| **Starting Page** | `/platform/{chatbotId}/workflows` |
| **Persona** | Customer success lead who wants automated notifications |

## Description

Use a pre-built template to create an automated workflow

## Preconditions

- User is logged in
- Organization's plan allows workflows

## Steps

### Step 1

Navigate to workflows page and see Workflows/Runs/Templates tabs

### Step 2

Click Templates tab — see 6 templates with category filters (Alerting, Escalation, Reporting, Moderation, General)

## Expected Outcome

Templates available: Negative Sentiment Alert, Escalation on Frustration, Topic Spike Monitor, Chatbot Quality Gate, Hallucination Flag, Positive Feedback Tracker
