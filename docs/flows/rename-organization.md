# Rename organization

## Overview

| Property | Value |
|----------|-------|
| **Flow** | Rename organization |
| **Starting Page** | `/platform/settings` |
| **Persona** | Organization owner updating the org name |

## Description

Update the organization name in settings

## Preconditions

- User is logged in
- User has Owner role

## Steps

### Step 1

Navigate to /platform/settings and see org name field with 'Test' value

### Step 2

Edit name field and click Save

## Expected Outcome

Organization name is updated across the platform
