# Filter conversations

## Overview

| Property | Value |
|----------|-------|
| **Flow** | Filter conversations |
| **Starting Page** | `/platform/{chatbotId}/conversations` |
| **Persona** | Analyst looking for specific types of conversations |

## Description

Use search and filters to narrow conversation results

## Preconditions

- User is logged in
- Chatbot has conversations

## Steps

### Step 1

Conversations page has search box, Filter button, sortable columns (User, Email, Organization, Messages, Sentiment, Language, Plan, Last Message, Created)

## Expected Outcome

Table updates to show only conversations matching filter criteria
