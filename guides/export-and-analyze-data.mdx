title: "Export and analyze data"

---

OpenBat lets you export all conversation data as JSON for offline analysis, and import historical conversations from JSON or CSV files.

## Export conversations

<Steps>
  <Step title="Navigate to Import/Export">
    Go to your chatbot's **Settings** > **Import/Export Data** tab.
  </Step>
  <Step title="Select export format">
    Choose **JSON** from the format dropdown.
  </Step>
  <Step title="Include analysis results (optional)">
    Check the **Include analysis results** checkbox to include sentiment scores, behavior flags, quality flags, and other AI annotations in the export.

    Leave it unchecked if you only need the raw conversation messages.
  </Step>
  <Step title="Download the file">
    Click **Export**. The JSON file downloads to your computer.
  </Step>
</Steps>

<Note>
  Large exports may take time if your chatbot has thousands of conversations. The export includes all conversations within the current date range.
</Note>

## Import historical data

Import conversations from another tool or historical logs to analyze them with OpenBat.

<Steps>
  <Step title="Download the template">
    Click **Download template CSV** to get a file showing the expected data format and column structure. Use this as a reference when preparing your data.
  </Step>
  <Step title="Prepare your file">
    Format your data as JSON or CSV matching the template structure. Each row or object represents a conversation with its messages.
  </Step>
  <Step title="Upload and import">
    Click **Choose File**, select your prepared file, and click **Import**. Imported conversations appear in your conversations list and are queued for analysis.
  </Step>
</Steps>

<Warning>
  Make sure your file matches the expected schema. Download the CSV template first to verify the format. Invalid files or mismatched columns cause the import to fail.
</Warning>

## Use cases

| Scenario | What to do |
|----------|-----------|
| **Offline analysis** | Export with analysis results, process in Python/R/Excel |
| **Regular reporting** | Export monthly, track trends outside OpenBat |
| **Backup** | Export periodically for data redundancy |
| **Migration** | Import conversations from a previous analytics tool |
| **Historical analysis** | Import old conversation logs to analyze retroactively |
